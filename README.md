# MonitorWorkspace Community

Welcome to the **MonitorWorkspace** community hub! This repository hosts our public discussions, feature requests, and support channels.

## What is MonitorWorkspace?

MonitorWorkspace is a **Google Workspace monitoring and administration tool** for IT admins. It provides user directory management, group health monitoring, Gmail & Chat oversight, license management, and employee offboarding tools.
